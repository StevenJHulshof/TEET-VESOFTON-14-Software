status_t VGA_FX_bloomScreen ( bloomIntensity_t  bloomInt)

Adds a blooming effect to the VGA screen.

This function will call VGA_L_getPixelData() to check the surrounding pixels. After averaging the RGB components of the color, it will call the VGA_L_setPixelData() for processing.

Parameters
bloomIntThe intensity of the bloom effect defined in the enum bloomIntensity_t. Ranging 0 to 7.
Returns
Status of operation. Possible return values:
VGA_SUCCESS - Operation is success.
VGA_PIXEL_NOT_SET - Set pixel operation has failed.
status_t status = VGA_FX_bloomScreen(VGA_BLOOM_INT_5);
if(status != VGA_SUCCESS) {
return status;
}
