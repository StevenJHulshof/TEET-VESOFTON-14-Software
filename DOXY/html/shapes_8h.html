Detailed Description

This file contains the draw functions of the VGA lib shapes.

The following shapes can be created using this file: Line, triangle, square, pentagon, hexagon, heptagon, octagon, nonagon, decagon, ellipse and polygon.

status_t VGA_P_drawLine ( sPosition_t  endPointPos[2],
color_t  lineColor,
uint8_t  lineWeight 
)

Draws line on VGA screen.

This function will call VGA_L_setLineData() for processing. Can be used if the user requires a straight line on the screen.

Parameters
endPointPosEnd point positions of the line.
lineColorColor of the line. Ranging 0 to 255.
lineWeightWeight of the line in pixels. Ranging 1 to 255.
Returns
Status of operation. Possible return values:
VGA_SUCCESS - Operation is success.
VGA_PIXEL_NOT_SET - Set pixel operation has failed.
VGA_LINE_COL_OUT_OF_BOUNDS - Input line color is out of bounds. (0 to 255).
VGA_LINE_WEIGHT_OUT_OF_BOUNDS - Input line weight is out of bounds. (1 to 255).
sPosition_t endPoints[2] = {{50, 50}, {100, 100}};
status_t status = VGA_P_drawLine(endPoints, VGA_COL_BLUE, 4);
if(status != VGA_SUCCESS) {
return status;
}

Parameters

numberOfVerticesTotal number of polygon vertices. Must be the same as the array length of verticePos.
lineColorLine color of the polygon. Ranging 0 to 255.
fillColorFill color of the polygon. Ranging -1 to 255.
lineWeightWeight of the line. Ranging 1 to 255.
Returns
Status of operation. Possible return values:
VGA_SUCCESS - Operation is success.
VGA_PIXEL_NOT_SET - Set pixel operation has failed.
VGA_LINE_COL_OUT_OF_BOUNDS - Input line color is out of bounds. (0 to 255).
VGA_FILL_COL_OUT_OF_BOUNDS - Input fill color is out of bounds. (-1 to 255).
VGA_LINE_WEIGHT_OUT_OF_BOUNDS - Input line weight is out of bounds. (1 to 255).
sPosition_t verticePos[5] = {
{10, 10}, {20, 20}, {50, 50}, (30, 40}, {40, 80}
};
status_t status = VGA_P_drawPolygon( verticePos,
5,
VGA_COL_BLACK,
VGA_COL_GREEN,
2 );
if(status != VGA_SUCCESS) {
return status;
}

Draws primitive on VGA screen.

This functions will call VGA_L_setPrimitiveData() for processing. Can be used if the user requires a default primitive shape.

Parameters
centerPointPosCenter point position of the primitive. Contains x and y coordinates.
radiiRadius x and y of the primitive. Must be above 0.
rotationDegreesRotation of the primitive in degrees.
lineColorPrimitive frame line color. Ranging 0 to 255.
fillColorPrimitive fill color. Ranging -1 to 255.
lineWeightPrimitive frame line weight. Ranging 1 to 255.
primitiveShapePrimitive default shape as the primitiveShape_t enum defines.
Returns
Status of operation. Possible return values:
VGA_SUCCESS - Operation is success.
VGA_PIXEL_NOT_SET - Set pixel operation has failed.
VGA_RADII_OUT_OF_BOUNDS - Input radii is out of bounds. (radii > 0).
VGA_LINE_COL_OUT_OF_BOUNDS - Input line color is out of bounds. (0 to 255).
VGA_FILL_COL_OUT_OF_BOUNDS - Input fill color is out of bounds. (-1 to 255).
VGA_LINE_WEIGHT_OUT_OF_BOUNDS - Input line weight is out of bounds. (1 to 255).
sPosition_t centerPos = {160, 120};
sRadii_t radii = (30, 50};
status_t status = VGA_P_drawPrimitive( &centerPos,
&radii,
45,
VGA_COL_RED,
VGA_COL_TRANSPARENT,
3,
VGA_PENTAGON );
if(status != VGA_SUCCESS) {
return status;
}
