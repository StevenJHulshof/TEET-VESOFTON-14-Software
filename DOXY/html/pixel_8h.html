Detailed Description

This file contains the draw functions of the VGA lib pixels.

The functionalities range from drawing a single a pixel on the screen to filling the entire screen with a color and reading a single pixel color.

status_t VGA_P_drawPixel ( sPosition_t *  pixelPos,
color_t  color 
)

Draws pixel on VGA screen.

This function will call the VGA_L_setPixelData() for processing. Can be used if the user requires a single pixel operation.

Parameters
pixelPosPosition of the pixel. Contains x and y coordinates.
colorColor of the pixel. Ranging 0 to 255.
Returns
Status of operation. Possible return values:
VGA_SUCCESS - Operation is success.
VGA_PIXEL_NOT_SET - Set pixel operation has failed.
VGA_PIXEL_OUT_OF_SCREEN - Input pixel position is outside screen bounds.
VGA_LINE_COL_OUT_OF_BOUNDS - Input line color is out of bounds. (0 to 255).
sPosition_t pixelPos = {100, 100};
status_t status = VGA_P_drawPixel(&pixelPos, VGA_COL_RED);
if(status != VGA_SUCCESS) {
return status;
}
status_t VGA_P_fillScreen ( color_t  color)

Fills screen with input color.

This function will call VGA_L_processScreenData() for processing. Can be used if the user requires a filled screen consisting of one color.

Parameters
colorColor of the screen fill. Ranging 0 to 255.
Returns
Status of operation. Possible return values:
VGA_SUCCESS - Operation is success.
VGA_PIXEL_NOT_SET - Set pixel operation has failed.
status_t status = VGA_P_fillScreen(VGA_COL_BLUE);
if(status != VGA_SUCCESS) {
return status;
}
